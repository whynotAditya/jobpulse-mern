# 🚀 JobPulse – Full Stack Job Portal

## 📌 Overview

JobPulse is a full stack job management application built using the MERN stack.
It allows users to create, view, and delete job listings with a clean and responsive UI.

---

## ⚙️ Tech Stack

* Frontend: React (Vite) + Axios
* Backend: Node.js + Express
* Database: MongoDB (Mongoose)

---

## ✨ Features

* ➕ Add new job listings
* 📄 View all jobs
* ❌ Delete jobs
* 🔄 Real-time updates after actions
* 🌐 REST API integration

---
